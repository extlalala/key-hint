Key Hint - IntelliJ IDEA 插件：实时快捷键视觉提示

Key Hint 是一个为 IntelliJ IDEA 开发的轻量级插件，旨在通过非侵入式视觉提示帮助开发者快速掌握常用快捷键。

功能特点：
在编辑代码时，自动弹出悬浮窗口，显示当前上下文可用的快捷键组合及其描述。
仅提供视觉提示，不会拦截或干扰任何键盘/鼠标操作，确保流畅的编码体验。
适用场景：
新手快速熟悉 IDE 快捷键。

Key Hint - IntelliJ IDEA Plugin: Real-time Keyboard Shortcut Visual Hints

Key Hint is a lightweight IntelliJ IDEA plugin designed to help developers master shortcuts faster through non-intrusive visual cues.

Features:
Auto-displays a floating hint window during coding, showing available keyboard shortcuts and their descriptions for the current context.
Purely visual hints with no keyboard/mouse event interception, ensuring uninterrupted workflow.
Use Cases:
Accelerate shortcut learning for beginners.
